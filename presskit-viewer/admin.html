JSON İçe Aktar

Mevcut press kit admin panelinden ("JSON Dışa Aktar" ile) aldığınız JSON dosyasını buraya yükleyin. Görsellerin URL'lerinin tam adres (örn. https://raw.githubusercontent.com/...) olması gerekir.

📄 JSON dosyasını buraya sürükleyin ya da tıklayıp seçin

İçe Aktarılan Veri:

Önizlemeyi kontrol edin ve bu tarayıcıda kalıcı olması için "Kaydet"e basın. Sitenin TÜM ziyaretçilere aynı içeriği göstermesi için ayrıca "data.json indir" ile aldığınız dosyayı presskit-viewer/data.json olarak repoya koyup deploy edin.
