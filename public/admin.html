GitHub Ayarları (sunucu yokken Kaydet/Yükle için)

Bu panel yerel sunucuya (localhost) bağlanamazsa — örn. GitHub Pages/Netlify'da statik olarak açıldığında — Kaydet ve veri yükleme burada girdiğiniz repoya doğrudan GitHub API üzerinden yapılır. Token'ı sadece bu repoya özel, "Contents: Read and write" izinli bir fine-grained Personal Access Token olarak oluşturun; sadece bu tarayıcıda (localStorage) saklanır, başka hiçbir yere gönderilmez.
